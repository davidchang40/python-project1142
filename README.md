# 專案介紹
在這個專案中，我們設計了一款以python為基底的統計計算機，可以讓使用者自行選擇要計算的項目，進行統計計算並輸出結果

### 專案目標
- 幫助使用者進行統計計算
- 讓使用者能自行選擇計算目標
- 能在輸入數據後得到結果

### 可選計算項目
- 信賴區間計算
- 假說檢定計算
- 平均數計算
- 變異數計算
- 標準差計算

# 程式碼功能介紹

### 主程式
利用def()語法，創建一個main()函式，並利用input()跟if語法，讓使用者能自行選擇不同的運算子函式運行，同時也能讓程式碼只運行被選擇的子函式部分，降低運算時間
 
### 子函式---平均數
利用input().split()接收使用者輸入的數據後，利用for迴圈進行計算，最後return結果

### 子函式---變異數和標準差
- 變異數:
先詢問使用者要進行母體還是樣本變異數後，再run一次上述的平均數函式後，利用if語法判斷對應的攻勢後，利用得到的數據進行再進行一次for迴圈計算再return回傳結果
- 標準差:
先run一遍上述的變異數函式，將回傳的數字開根號

### 子函式---信賴區間
先利用dict()字典語法儲存計算會用到的數據，再利用def()和if語法撰寫確認並取用字典裡的值，之後用def()語法建立一個confident_interval()的子計算函式，用input()、if和基礎運算語法來接收使用者輸入之數據、判斷運用公式、計算並回傳結果

### 子函式 — 假說檢定run_test
為假設檢定區的主控函式，負責呼叫計算區函式及 check_hypothesis()。
先利用全域變數區宣告所有會用到的變數，包含用來儲存用戶條件的 binary/ternary 形變數（如檢定類型、尾數、信心水準等），以及用來儲存原始數據、統計量的 list 和變數。
執行時先調用 _reset_hypothesis_globals() 清洗上次殘留的資料，再使用 ask() 配合問答流程，以字典語法將用戶的輸入對應到條件變數，最後根據條件組合（是否為比例型、母體是否已知、樣本數、是否配對）決定要 run 哪一個計算函式。

### 子函式 — 假說檢定計算函式區
共定義五個函式，分別對應單樣本 Z 檢定、雙樣本 Z 檢定、單樣本 T 檢定、獨立雙樣本 T 檢定及配對 T 檢定。
各函式引用 x_bar、sigma、n_ht 等全域變數進行計算，並在函式內先判斷用戶是否輸入原始數據——若是，則先利用 NumPy 方法將原始數據純化為所需統計量，再藉由公式及 SciPy 方法計算並回傳檢定統計量。

### 子函式 — 假說檢定判定區
由 check_hypothesis() 接收來自計算函式的統計量，利用 SciPy 方法查出對應的臨界值，再透過 if...elif...else 判斷左尾、右尾或雙尾的拒絕條件，最終輸出統計量、臨界值與是否拒絕 H₀ 的結論。
