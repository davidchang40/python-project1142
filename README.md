# 專案介紹
在這個專案中，我們設計了一款以python為基底的統計計算機，可以讓使用者自行選擇要計算的項目，進行統計計算並輸出結果

### 專案目標
- 幫助使用者進行統計計算
- 讓使用者能自行選擇計算目標
- 能在輸入數據後得到結果

### 可選計算項目
- 信賴區間計算
- 假說檢定計算
- 平均數計算
- 變異數計算
- 標準差計算

# 程式碼功能介紹

### 主程式
利用def()語法，創建一個main()函式，並利用input()跟if語法，讓使用者能自行選擇不同的運算子函式運行，同時也能讓程式碼只運行被選擇的子函式部分，降低運算時間
 
### 子函式---平均數
利用input().split()接收使用者輸入的數據後，利用for迴圈進行計算，最後return結果

### 子函式---變異數和標準差
- 變異數:
先詢問使用者要進行母體還是樣本變異數後，再run一次上述的平均數函式後，利用if語法判斷對應的攻勢後，利用得到的數據進行再進行一次for迴圈計算再return回傳結果
- 標準差:
先run一遍上述的變異數函式，將回傳的數字開根號

### 子函式---信賴區間
先利用dict()字典語法儲存計算會用到的數據，再利用def()和if語法撰寫確認並取用字典裡的值，之後用def()語法建立一個confident_interval()的子計算函式，用input()、if和基礎運算語法來接收使用者輸入之數據、判斷運用公式、計算並回傳結果
